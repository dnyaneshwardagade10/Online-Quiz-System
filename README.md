# Online Quiz System & Student Management System

This repository contains two separate projects:

## 1. Online Quiz App
Located in `online-quiz-app/`.
The original quiz system.
- **Backend**: Node.js/Express
- **Database**: MySQL (`online_quiz_db`)

## 2. Student Management System
Located in `student-management-system/`.
A new Enterprise CRUD application.
- **Backend**: Node.js/Express
- **Database**: MySQL (`student_management_db`)
- **Features**: Full CRUD, Search, Sort, Validation.

## Setup
Each project is independent. Navigate to the respective folder and run `npm install` && `npm start`.
