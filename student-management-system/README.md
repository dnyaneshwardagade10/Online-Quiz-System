# Student Management System

A separate, Enterprise-grade CRUD application for managing students.

## Features
- Full CRUD operations for Students
- MySQL Database with Relational Integrity
- MVC Architecture (Node.js + Express)
- Search and Sorting
- Frontend with Vanilla JS/CSS

## Setup
1.  Navigate to this folder: `cd student-management-system`
2.  Install dependencies: `npm install`
3.  Start the server: `npm start`
4.  Open `http://localhost:3000` in your browser.

## Database
The system uses a MySQL database named `student_management_db`.
Credentials are configured in `.env`.
The database and tables are automatically created on server start.
